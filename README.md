## 🧠 Bagaimana Minimax Bekerja?

Algoritma Minimax adalah algoritma pencarian rekursif yang sering digunakan dalam teori permainan (*game theory*) dan AI. Algoritma ini memvisualisasikan semua kemungkinan langkah ke depan dalam bentuk *tree*.

1. **Maximizer (AI):** Berusaha mendapatkan skor tertinggi (+10 untuk menang).
2. **Minimizer (Pemain):** AI berasumsi pemain akan bermain optimal dan berusaha memberikan skor terendah (-10 jika pemain menang).
3. **Draw:** Jika papan penuh tanpa pemenang, skornya adalah 0.

AI akan selalu memilih langkah dengan nilai evaluasi tertinggi dari *tree* kemungkinan tersebut.
